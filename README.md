# WALCHAND-IMAGINE-CUP-2026
Walchand Imagine Cup 2026 (WIC 26) is a national-level innovation and startup challenge organized by the Microsoft Learn Students’ Club (MLSC), Walchand College of Engineering, Sangli. Inspired by the global Microsoft Imagine Cup, the event encourages students to ideate, build, and pitch technology-based solutions to real-world problems.
